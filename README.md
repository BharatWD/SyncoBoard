# 🚀 SyncoBoard

A real-time collaborative Kanban board built with React, TypeScript, Express, and WebSockets. SyncoBoard helps teams organize tasks, track progress, and collaborate seamlessly in real time.

## ✨ Features

* 📋 Kanban-style task management
* 🤝 Real-time collaboration
* 🔄 Drag-and-drop task workflow
* 👥 Team member assignments
* 💬 Task comments and discussions
* 📊 Board insights and analytics
* ⚡ WebSocket-powered live updates
* 🎨 Modern responsive UI

## 🛠 Tech Stack

### Frontend

* React 19
* TypeScript
* Vite
* Zustand
* Tailwind CSS

### Backend

* Node.js
* Express.js
* WebSocket (ws)

### Visualization

* D3.js

## 🚀 Getting Started

### Installation

```bash
git clone https://github.com/BharatWD/SyncoBoard.git
cd SyncoBoard
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
